# 《计算机程序设计基础》实验报告答案

**本仓库的《计算机程序设计基础》实验报告对应的课程时间为 2023 ~ 2024 下半学年。**

命名对应着题目，如 `1.cpp` 对应该实验第一题。部分非纯文本题目在文件夹下面的 `.md` 文档中。

本仓库的内容，除了 MFC 编程题，其他都是源代码，需要自己在 VS2010 里面创建项目，然后复制粘贴代码到 VS2010 里面才能运行。

欢迎各位大佬批评指正。
